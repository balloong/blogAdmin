## 构建项目

执行以下命令创建项目

```swift
npm init vite@latest easyblog-admin
```

## 初始化项目

进入到 创建的项目 cd easyblog-admin 执行以下命令

```undefined
npm install
```

# 运行

```undefined
npm run dev
```

没报错的话，说明项目已经成功构建
